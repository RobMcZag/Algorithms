The Eclipse files (.project, .classpath and .settings) have been removed.

The project files have been setup according to a default Maven/Gradle layout. JUnit v4 is defined as a dependency, and will be downloaded from MavenCentral by Gradle.

In order to import the project in Eclipse

Install Help/Eclise Marketplace Gradle Integration for Eclipse (4.4) 3.6.1 RELEASE ignoring the warnings.
Import the project with File/Import, select Gradle Project, select the directory root
click on Build Model
From the console,

gradle build builds the jar
gradle test to run tests.

You'll find one passing test, which you can run from Eclipse or from gradle.

There's only 1 .gitignore file.
